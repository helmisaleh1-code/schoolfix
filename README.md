# SchoolFix

SchoolFix adalah website startup teknologi berbasis Laravel yang membantu sekolah dalam melaporkan dan mengelola kerusakan fasilitas secara digital.

## Deskripsi Project
Project ini dibuat untuk menyelesaikan masalah pelaporan kerusakan fasilitas sekolah yang masih dilakukan secara manual. Dengan SchoolFix, laporan kerusakan dapat dicatat, dipantau, dan dikelola dengan lebih cepat dan terstruktur.

## Permasalahan
Di banyak sekolah, kerusakan seperti komputer lab rusak, proyektor mati, lampu kelas mati, dan fasilitas lainnya sering terlambat ditangani karena proses pelaporan masih manual.

## Solusi
SchoolFix menyediakan sistem berbasis web untuk:
- mencatat laporan kerusakan
- mengelola status laporan
- memantau proses perbaikan
- menyimpan data laporan secara rapi

## Fitur Utama
- Dashboard laporan
- Tambah laporan
- Edit laporan
- Hapus laporan
- Status laporan: Menunggu, Diproses, Selesai

## Cara Menjalankan Project
1. Clone repository ini
2. Jalankan `composer install`
3. Copy file `.env.example` menjadi `.env`
4. Atur koneksi database pada file `.env`
5. Jalankan `php artisan key:generate`
6. Jalankan `php artisan migrate --seed`
7. Jalankan `php artisan serve`
